# SignSystem
用户报名系统设计
  本系统场景设计为2020级新入学的本校学生秒杀某志愿者活动报名的业务。
  - .sql 为项目使用的数据库文件
  - Receive文件夹中的php文件为服务器运行接收rabbitMQ消息并写入数据库的程序
  - 172.81.214.81文件夹为活动报名系统程序
  - www.pkuyang.design文件夹为后台管理系统程序
  
环境配置：
  1. Centos 7.6系统
	2. PHP 7.2
	3. MYSQL 5.6
	4. Thinkphp 5
	5. redis 6.0.9
	6. Apache 2.4.46
	7. Memcached 1.6.9
